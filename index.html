      Fast Length Adjustment of Short reads

      An accurate and a fast tool to merge paired-end reads that were generated from DNA fragments whose lengths are shorter than twice the length of reads.

      Read the FLASH manual for more info
      minoverlap

      Set the minimum required overlap length between two reads to provide a confident overlap.


      mismatchRatio

      Set the maximum allowed ratio of the number of mismatches at the overlap length.


      maxOverlap

      Set the maximum overlap length expected in approximately 90% of read pairs.

      read length.

      Average read length.



      fragment length

      Average fragment length



      standard deviation of fragment lengths

      f you do not know standard deviation of the fragment library, you can probably assume that the standard deviation is 10% of the average fragment length



      phredOffset

      Set the smallest ASCII value of the characters used to represent quality values of bases in fastq files. 33 for latest Illumina and Sanger or 64 for earlier Illumina.
